# Email Drafting Skill

When drafting emails:

## Structure
1. **Subject Line**: Clear, specific, action-oriented (not "Hello" or "Question")
2. **Opening**: Context + purpose in 1-2 sentences
3. **Body**: Key points in order of importance; use bullets for multiple items
4. **Call to Action**: What you need from the recipient and by when
5. **Closing**: Appropriate sign-off matching the relationship

## Tone Guidelines
- **Professional**: Default for work contexts. Clear, respectful, concise.
- **Friendly**: For colleagues you know well. Warmer but still purposeful.
- **Formal**: For executives, external partners, or sensitive topics.

## Best Practices
- One email = one topic. Multiple topics → multiple emails.
- Keep it under 5 short paragraphs
- Bold or bullet key dates, numbers, or action items
- If replying, address all points from the original email
- Proofread for tone — remove unnecessary hedging ("just", "maybe", "I think perhaps")
