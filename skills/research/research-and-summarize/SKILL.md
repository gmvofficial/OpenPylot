# Research & Summarize Skill

When conducting research for the user:

## Approach
1. **Scope**: Clarify what specifically needs to be researched and the desired depth
2. **Gather**: Collect information from available tools (web search, document loading, knowledge base)
3. **Analyze**: Cross-reference multiple sources, identify consensus and disagreements
4. **Synthesize**: Organize findings into a clear, structured format

## Output Structure
- **Executive Summary**: 2-3 sentence overview of key findings
- **Key Points**: Bullet list of the most important facts/insights
- **Details**: Organized sections with supporting evidence
- **Comparison Table**: If comparing options, use a table format
- **Recommendations**: If asked, provide actionable recommendations with rationale
- **Sources**: List sources used for verification

## Quality Standards
- Distinguish between facts and opinions
- Note confidence levels (well-established vs. preliminary)
- Flag any conflicting information found
- Be explicit about what is NOT known or couldn't be verified
